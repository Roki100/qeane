Qeane, a very good multi-purpose bot easy to use!

What is Qeane? It's a bot that has multiples features like moderation, fun commands and more! To get a list of commands, just type qeane help

This bot also offers tags, and here is how they work:
First, You create a tag with the tag-create command, like this: qeane tag-create sleep Please sleep more!

Second, you use the tag with qeane sleep and the bot will respond with "Please sleep more"! It's that easy! The tag-delete and tag-list commands are self-explanatory.


Tags are just one of the many features Qeane has, so dont don't hesitate to try it out!
